# Paramour
The perfect rendezvous with JavaScript and Python
----
### How to:
Simply pass a string to the Paramour function, and it'll return a parsable string
----
# Awesome Features of Paramour
### Functions
```javascript
function_name(arguments) {
  ...
}
```

### Classes
```python
class_name {
  <init>(arguments) {
    # constructor
    ...
  }
}

super_name.class_name {
  <init> {
    # optional arguments
    ...
  }
}
```

### Tuples
```javascript
(variable_name) = {{
  123,
  "abc",
  /def/g
}};
```
